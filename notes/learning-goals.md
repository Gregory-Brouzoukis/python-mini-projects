# Learning Goals

This file keeps the main learning goals for this Python practice repository.

## Main goals

1. Write code more often
2. Understand Python basics better
3. Build small projects from start to finish
4. Practise problem solving
5. Learn how to organize code in GitHub
6. Improve explanations and documentation

## What every project should include

1. A clear purpose
2. Simple and readable code
3. Good variable names
4. Comments only where they help
5. A short explanation
6. Possible future improvements

## Long term goal

The long term goal is to use these small projects as a base for bigger projects in web development, applications and AI automation.
